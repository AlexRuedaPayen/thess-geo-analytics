# thess-geo-analytics

### Environement Setup

This project requries Python 3.11

#### 1. Create a virtual environement

```bash
python -m venv .venv
```

#### 2. Add the virtual environement to .gitignore

```
.venv/
```

#### 3. Install dependencies

```bash
.\.venv\Scripts\pip.exe install -r requirements.txt
```
